1补充

2枚举基本用法

  • enum case; 一般枚举的名字大写，枚举项小写；
  • 关联值
  • 原始值 定义枚举的时候指定类型，定义枚举项的时候定义 默认值，是原始值
  • 隐式原始值;如果 原始值类型是Int，String，swift会自动分配原始值
  • 递归枚举 需要在枚举name之前，或者需要递归的枚举项之前 加上关键字 indirect

  • 枚举占用的内存

    • 使用MemoryLayout 来获取数据类型占用的内存大小

    • 枚举有关联值，有原始值 是不一样的内存大小，有区别。

3 可选项

  • 可选绑定

    • while lt num = Int("323a"),num < 0 {}

  • 空合并运算符??

  • 隐式解包，在定义类型后面 加上感叹号!

  • 多重可选项
