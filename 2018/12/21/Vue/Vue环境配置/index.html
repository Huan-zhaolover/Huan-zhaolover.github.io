Vue环境配置

需要配置有node / npm. 终端查看node -v npm -v 查看是否有，如果没有需要提前安装

  • vue -V 查看是否安装Vue

  • npm install -g @vue/cli 安装Vue脚手架

  • vue create hello-world 创建项目

  • npm run dev 开发测试

  • npm run build 打包
